DICCIONARIO DE TÉRMINOS

Contenido DUA

-Breakout: es un juego educativo al estilo de un escape room, físico o digital, en el que el alumnado (u otras personas) debe resolver retos, acertijos o pruebas, en orden, para “abrir cajas”, conseguir códigos o “escapar”.

-Conectores: en lingüística son palabras o expresiones que unen ideas o frases, mostrando relaciones lógicas como causa, consecuencia, tiempo, contraste, etc.

-DUA: abreviatura para “diseño universal para el aprendizaje”.

-Infografía: es un recurso visual que combina texto + imágenes + iconos + datos + otros elementos (incluidos interactivos) para explicar de manera clara y atractiva una información compleja.

-Partizip II: es el equivalente al participio pasado en castellano; es la forma del verbo que se usa para formar tiempos compuestos como el Perfekt, el Plusquamperfekt o la voz pasiva.

-Perfekt: un tiempo verbal del alemán que se usa para expresar acciones pasadas; es muy usado en el lenguaje oral. Se forma con verbo auxiliar+participio II.

-Podcast: es una grabación de audio, normalmente en formato conversación o exposición, que se puede escuchar en línea o descargar.

-Präteritum: un tiempo verbal del alemán que se usa para expresar acciones pasadas; es usado en lenguaje escrito. Los verbos regulares lo forman “raíz del verbo+te”.

-SdA: abreviatura para “situación de aprendizaje”.

-Verbo auxiliar: es un verbo que acompaña al verbo principal para formar tiempos compuestos. En alemán, los auxiliares son: “haben” y “sein” (para cambios espaciales o cambios de estado.)
